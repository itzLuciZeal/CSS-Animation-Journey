# 🎨 Simple Mini-Projects Overview

This directory serves as my experimental playground. Each sub-folder represents a specific technical challenge where I break down complex UI trends and reconstruct them using raw HTML and CSS.

## 🚀 Current Projects

### [01-CSS-Loaders](./01-CSS-Loaders/)
**Focus:** Advanced Animation & Masking.
* **Techniques Used**: `-webkit-mask-composite`, `box-shadow` layering, and Keyframe transformations.
* **Technical Highlight**: Created a "Dotted Ring" loader using a single `div` and strategic `box-shadow` coordinates to minimize DOM elements.

### [02-Profile-Cards](./02-Profile-Cards/) (In Progress)
**Focus:** Glassmorphism & Depth.
* **Goal**: Master `backdrop-filter` and `rgba` transparency to create modern, frosted-glass UI components.

## 🧠 The "Micro-Learning" Philosophy

In my path toward becoming a **Data Scientist**, I’ve realized that complex systems are just collections of simple, well-organized parts. These mini-projects allow me to:
1. **Isolate Variables**: Perfecting a single animation before scaling it.
2. **Build a Component Library**: These loaders will be integrated into my **Major Projects**, like the Math Tutoring platform.
3. **Architectural Discipline**: Ensuring even practice files follow professional folder structures.
