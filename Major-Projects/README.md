# 🏗️ Major Projects Overview

This directory contains full-scale web applications and platforms. Unlike the mini-projects, these files focus on solving real-world problems, integrating backend logic, and providing a complete user experience.

## 🌟 Featured Projects

### [📐 Math Tutoring Business Platform](./01-Math-Tutoring/)
**Status:** In Development  
A comprehensive personal business website for selling PDF lessons and managing tutoring services (online or face-to-face).
* **Key Features**: Professional landing page with integrated "Cheat Sheets," email-based booking system, and a PDF storefront.
* **Tech Stack**: HTML5, Advanced CSS (Animations/Glassmorphism), JavaScript, and planned Python/FastAPI integration for secure payment and database management.

### [🚀 Personal Portfolio & Data Science Hub](./02-Portfolio-Data-Hub/)
**Status:** Planning/Architecture  
My central command center where I showcase my projects, host interactive data science dashboards, and share technical knowledge.
* **Key Features**: Live Data Science visualizations using Pandas, a "Project of the Day" log, and a technical blog for knowledge sharing.
* **Backend Focus**: Implementing **SQLite** for data storage and **FastAPI** for high-performance handling of data-heavy dashboards.

## 🛠️ Engineering Standards

Each major project in this repository adheres to the following professional standards:

1.  **Component Reusability**: Utilizing the animations and loaders perfected in the `Simple-Mini-Projects` directory to ensure a consistent, high-end user experience.
2.  **Security First**: Implementing robust authentication logic, including password hashing and salts, to protect user and business data.
3.  **Data Integrity**: Using visual database tools to inspect data flow and ensure structural stability for both tutoring records and data dashboards.
